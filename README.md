# Quaero

Quaero is a product discovery studio.

Its purpose is not to build products.

Its purpose is to discover which products deserve to be built.

Ideas compete for evidence.
Reality decides.

## Principles

- Seek before building
- Test before investing
- Evidence before conviction

## Technical Goals

- Fast iteration
- Simple deployment
- Static-first
- No CMS
- No database unless necessary
- No authentication unless necessary
